# Masked Avatars

A stunning avatar stack with masked overlapping effects and animated text rings on hover. Each avatar reveals the person's name in a circular pattern when hovered.

<ComponentPreview
  component={
    <div className="relative w-full h-[300px] rounded-lg overflow-hidden bg-neutral-100 dark:bg-zinc-900 flex items-center justify-center">
      <MaskedAvatars />
    </div>
  }
  code={`import MaskedAvatars from '@/components/ui/masked-avatars'

export function Demo() {
  const avatars = [
    { avatar: 'https://i.pravatar.cc/150?u=1', name: 'Emma' },
    { avatar: 'https://i.pravatar.cc/150?u=2', name: 'James' },
    { avatar: 'https://i.pravatar.cc/150?u=3', name: 'Sophie' },
  ]
  
  return <MaskedAvatars avatars={avatars} />
}`}
/>
