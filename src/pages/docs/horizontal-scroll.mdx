# Horizontal Scroll

A smooth horizontal scrolling section that transforms vertical scroll into horizontal movement. Perfect for portfolios, galleries, and feature showcases.

  return (

            {item.content}
          </div>

      </div>
    </HorizontalScroll>
  )
}`}
/>
