# CLI

Use the Obsidian UI CLI to quickly add components to your project.

---

## Usage

Run the following command to add a component:

<CodeBlock 
  code="npx obsidian-ui add [component-name]"
  language="bash"
/>

## Example

To add the animated button component:

<CodeBlock 
  code="npx obsidian-ui add animated-hero"
  language="bash"
/>

## Available Commands

| Command | Description |
|---------|-------------|
| `add` | Add a component to your project |
| `init` | Initialize Obsidian UI in your project |
| `list` | List all available components |

---

## MCP Server

The shadcn MCP Server allows AI assistants to interact with items from registries. You can browse available components, search for specific ones, and install them directly into your project using natural language.

### Configuration

Add the Obsidian UI registry to your project's `components.json` file.

<CodeBlock 
  code={`{
  "registries": {
    "@obsidian": "https://www.obsidianui.com/r/{name}.json"
  }
}`}
  language="json"
/>

### Initialization

Add the MCP server to your project:

<CodeBlock 
  code="npx shadcn@latest mcp init"
  language="bash"
/>

### Client Setup

Configure for your AI assistant (e.g., Cursor):

<CodeBlock 
  code="npx shadcn@latest mcp init --client cursor"
  language="bash"
/>
