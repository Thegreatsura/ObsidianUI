# Leave Rating

An interactive star rating component with an animated emoji face that changes expression based on the selected rating.

## Usage

<CodeBlock language="tsx" code={`import { LeaveRating } from "@/components/ui/leave-rating"

export function Demo() {
  return (
    <LeaveRating
      question="How was your experience?"
      buttonText="Submit"
      onSubmit={(rating) => console.log("Rating:", rating)}
    />
  )
}`} />
