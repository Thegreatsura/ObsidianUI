# 3D Displacement Text

An interactive WebGL text effect with 3D displacement that responds to mouse movement. Creates a liquid, morphing effect on the text.

  component={
    <div className="relative w-full h-[450px] rounded-lg overflow-hidden bg-gradient-to-br from-neutral-900 via-neutral-800 to-neutral-900 flex items-center justify-center">
      <LiquidText text="MORPH" fontSize={140} color="#ffffff" className="h-full" />
    </div>
  }
  code={`import LiquidText from '@/components/ui/liquid-text'

export function Demo() {
  return <LiquidText text="MORPH" fontSize={140} color="#ffffff" />
}`}
/>
